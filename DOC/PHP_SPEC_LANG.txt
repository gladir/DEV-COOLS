==============================================================================
  SPECIFICATIONS DU LANGAGE DE PROGRAMMATION PHP
==============================================================================

  PHP (PHP: Hypertext Preprocessor) est un langage de programmation
  interprete, a typage dynamique, principalement concu pour le
  developpement web cote serveur. Il supporte la programmation
  procedurale, orientee objet et fonctionnelle.

  Version de reference : PHP 8.x

==============================================================================
  1. SYNTAXE DE BASE
==============================================================================

  1.1 Balises PHP
  -----------------------------------------

  Le code PHP est delimite par des balises :

    <?php ... ?>          Balise standard (recommandee)
    <?= expr ?>           Balise courte d'affichage (echo)

  1.2 Instructions et separateurs
  -----------------------------------------

  Chaque instruction se termine par un point-virgule :

    $x = 10;
    echo "Bonjour";

  1.3 Commentaires
  -----------------------------------------

    // Commentaire sur une ligne
    # Commentaire sur une ligne (style shell)
    /* Commentaire
       sur plusieurs lignes */
    /** Commentaire de documentation (PHPDoc) */

  1.4 Sensibilite a la casse
  -----------------------------------------

  Les noms de variables sont sensibles a la casse :

    $nom et $Nom sont deux variables differentes.

  Les mots-clefs, noms de fonctions et noms de classes ne sont
  pas sensibles a la casse :

    echo, ECHO, Echo sont equivalents.

==============================================================================
  2. TYPES DE DONNEES
==============================================================================

  2.1 Types scalaires
  -----------------------------------------

  Type         Description                    Exemple
  -------      ---------------------------    ----------------
  bool         Booleen                        true, false
  int          Entier                         42, 0x1A, 0b1010, 0o17
  float        Nombre a virgule flottante     3.14, 1.2e3
  string       Chaine de caracteres           "texte", 'texte'

  2.2 Types composes
  -----------------------------------------

  Type         Description
  -------      ---------------------------
  array        Tableau ordonne / associatif
  object       Instance de classe
  callable     Fonction appelable
  iterable     Tableau ou objet Traversable

  2.3 Types speciaux
  -----------------------------------------

  Type         Description
  -------      ---------------------------
  resource     Reference a une ressource externe
  null         Absence de valeur
  never        Fonction qui ne retourne jamais (PHP 8.1)
  void         Fonction sans valeur de retour
  mixed        Tout type (PHP 8.0)

  2.4 Verification de type
  -----------------------------------------

    gettype($var)            Retourne le type sous forme de chaine
    is_int($var)             Verifie si entier
    is_float($var)           Verifie si flottant
    is_string($var)          Verifie si chaine
    is_bool($var)            Verifie si booleen
    is_array($var)           Verifie si tableau
    is_object($var)          Verifie si objet
    is_null($var)            Verifie si null
    is_numeric($var)         Verifie si numerique
    isset($var)              Verifie si defini et non null
    empty($var)              Verifie si vide
    var_dump($var)           Affiche le type et la valeur

  2.5 Transtypage (cast)
  -----------------------------------------

    (int) $var               Convertir en entier
    (float) $var             Convertir en flottant
    (string) $var            Convertir en chaine
    (bool) $var              Convertir en booleen
    (array) $var             Convertir en tableau
    (object) $var            Convertir en objet
    intval($var)             Convertir en entier (fonction)
    floatval($var)           Convertir en flottant (fonction)
    strval($var)             Convertir en chaine (fonction)
    settype($var, "type")    Changer le type de la variable

==============================================================================
  3. VARIABLES ET CONSTANTES
==============================================================================

  3.1 Variables
  -----------------------------------------

  Les variables commencent par le signe dollar ($) :

    $nom = "PHP";
    $age = 30;
    $pi  = 3.14159;

  Les noms de variables doivent commencer par une lettre ou un
  underscore, suivis de lettres, chiffres ou underscores.

  3.2 Variables variables
  -----------------------------------------

    $nom = "langage";
    $$nom = "PHP";       // equivalent a $langage = "PHP"

  3.3 Variables predefinies (superglobales)
  -----------------------------------------

  Variable         Description
  -------          -------------------------------------------
  $GLOBALS         Toutes les variables globales
  $_SERVER         Informations du serveur et de l'execution
  $_GET            Variables HTTP GET
  $_POST           Variables HTTP POST
  $_FILES          Fichiers envoyes par HTTP
  $_COOKIE         Cookies HTTP
  $_SESSION        Variables de session
  $_REQUEST        Contenu de $_GET, $_POST et $_COOKIE
  $_ENV            Variables d'environnement

  3.4 Constantes
  -----------------------------------------

    define("NOM", valeur);           Definir une constante
    const NOM = valeur;              Definir une constante (classe/global)

  Constantes predefinies :

    PHP_VERSION       Version de PHP
    PHP_INT_MAX       Entier maximum
    PHP_INT_MIN       Entier minimum
    PHP_INT_SIZE      Taille d'un entier en octets
    PHP_FLOAT_MAX     Flottant maximum
    PHP_FLOAT_MIN     Flottant minimum positif
    PHP_EOL           Fin de ligne du systeme
    PHP_OS            Systeme d'exploitation
    PHP_SAPI          Interface serveur utilisee
    DIRECTORY_SEPARATOR  Separateur de repertoire (/ ou \)
    PATH_SEPARATOR    Separateur de chemin (: ou ;)
    TRUE              Booleen vrai
    FALSE             Booleen faux
    NULL              Valeur nulle
    PHP_MAJOR_VERSION Version majeure
    PHP_MINOR_VERSION Version mineure
    INF               Infini
    NAN               Not a Number
    E_ERROR           Erreur fatale
    E_WARNING         Avertissement
    E_NOTICE          Notification
    E_DEPRECATED      Fonctionnalite obsolete

  3.5 Portee des variables
  -----------------------------------------

  Par defaut, les variables ont une portee locale a la fonction.
  Pour acceder a une variable globale dans une fonction :

    $x = 10;
    function test() {
        global $x;       // Acces a la variable globale
        echo $x;
    }

  Le mot-clef static conserve la valeur entre les appels :

    function compteur() {
        static $n = 0;
        $n++;
        return $n;
    }

  4. OPERATEURS
==============================================================================

  4.1 Operateurs arithmetiques
  -----------------------------------------

  Operateur    Description             Exemple
  ---------    --------------------    -----------
  +            Addition                $a + $b
  -            Soustraction            $a - $b
  *            Multiplication          $a * $b
  /            Division                $a / $b
  %            Modulo                  $a % $b
  **           Exponentiation          $a ** $b

  4.2 Operateurs d'affectation
  -----------------------------------------

  Operateur    Description
  ---------    --------------------
  =            Affectation simple
  +=           Addition et affectation
  -=           Soustraction et affectation
  *=           Multiplication et affectation
  /=           Division et affectation
  %=           Modulo et affectation
  **=          Exponentiation et affectation
  .=           Concatenation et affectation
  ??=          Affectation si null (PHP 7.4)

  4.3 Operateurs de comparaison
  -----------------------------------------

  Operateur    Description                     Exemple
  ---------    ----------------------------    -----------
  ==           Egal (avec conversion)          $a == $b
  ===          Identique (type et valeur)      $a === $b
  !=           Different                       $a != $b
  <>           Different                       $a <> $b
  !==          Non identique                   $a !== $b
  <            Inferieur                       $a < $b
  >            Superieur                       $a > $b
  <=           Inferieur ou egal               $a <= $b
  >=           Superieur ou egal               $a >= $b
  <=>          Comparaison combinee            $a <=> $b

  L'operateur <=> (spaceship) retourne -1, 0 ou 1.

  4.4 Operateurs logiques
  -----------------------------------------

  Operateur    Description
  ---------    --------------------
  &&           ET logique
  ||           OU logique
  !            NON logique
  and          ET logique (priorite basse)
  or           OU logique (priorite basse)
  xor          OU exclusif

  4.5 Operateurs bit a bit
  -----------------------------------------

  Operateur    Description
  ---------    --------------------
  &            ET bit a bit
  |            OU bit a bit
  ^            XOR bit a bit
  ~            NON bit a bit
  <<           Decalage a gauche
  >>           Decalage a droite

  4.6 Operateurs de chaine
  -----------------------------------------

  Operateur    Description
  ---------    --------------------
  .            Concatenation
  .=           Concatenation et affectation

  4.7 Operateurs d'incrementation
  -----------------------------------------

  Operateur    Description
  ---------    --------------------
  ++$a         Pre-incrementation
  $a++         Post-incrementation
  --$a         Pre-decrementation
  $a--         Post-decrementation

  4.8 Operateurs speciaux
  -----------------------------------------

  Operateur    Description
  ---------    ----------------------------------------
  ?:           Operateur ternaire
  ??           Coalescence nulle (PHP 7.0)
  ?->          Appel de methode null-safe (PHP 8.0)
  @            Suppression d'erreur
  =>           Separateur clef/valeur dans un tableau
  ->           Acces a un membre d'objet
  ::           Operateur de resolution de portee
  ...          Operateur de decomposition (splat)
  instanceof   Verification de type d'objet

==============================================================================
  5. STRUCTURES DE CONTROLE
==============================================================================

  5.1 Conditions
  -----------------------------------------

  if / elseif / else :

    if ($a > $b) {
        echo "a est plus grand";
    } elseif ($a == $b) {
        echo "a est egal a b";
    } else {
        echo "b est plus grand";
    }

  switch :

    switch ($couleur) {
        case "rouge":
            echo "Rouge";
            break;
        case "bleu":
            echo "Bleu";
            break;
        default:
            echo "Autre";
            break;
    }

  match (PHP 8.0) :

    $resultat = match ($code) {
        200     => "OK",
        404     => "Non trouve",
        500     => "Erreur serveur",
        default => "Inconnu",
    };

  5.2 Boucles
  -----------------------------------------

  while :

    while ($i < 10) {
        echo $i;
        $i++;
    }

  do...while :

    do {
        echo $i;
        $i++;
    } while ($i < 10);

  for :

    for ($i = 0; $i < 10; $i++) {
        echo $i;
    }

  foreach :

    foreach ($tableau as $valeur) {
        echo $valeur;
    }

    foreach ($tableau as $clef => $valeur) {
        echo "$clef : $valeur";
    }

  5.3 Controle de boucle
  -----------------------------------------

    break;           Sortir de la boucle
    break N;         Sortir de N niveaux de boucles
    continue;        Passer a l'iteration suivante
    continue N;      Passer a l'iteration suivante de N niveaux

  5.4 Inclusion de fichiers
  -----------------------------------------

    include "fichier.php";         Inclure (avertissement si absent)
    include_once "fichier.php";    Inclure une seule fois
    require "fichier.php";         Inclure (erreur fatale si absent)
    require_once "fichier.php";    Inclure une seule fois (erreur fatale)

  5.5 Syntaxe alternative
  -----------------------------------------

  PHP supporte une syntaxe alternative pour les structures de
  controle, utile dans les gabarits HTML :

    if ($cond):
        ...
    elseif ($cond2):
        ...
    else:
        ...
    endif;

    while ($cond):
        ...
    endwhile;

    for (...):
        ...
    endfor;

    foreach ($arr as $v):
        ...
    endforeach;

    switch ($v):
        ...
    endswitch;

==============================================================================
  6. FONCTIONS
==============================================================================

  6.1 Declaration de fonction
  -----------------------------------------

    function nom($param1, $param2) {
        // corps de la fonction
        return $resultat;
    }

  6.2 Parametres par defaut
  -----------------------------------------

    function saluer($nom = "monde") {
        echo "Bonjour $nom";
    }

  6.3 Declaration de type (type hints)
  -----------------------------------------

    function additionner(int $a, int $b): int {
        return $a + $b;
    }

  Types acceptes pour les parametres et le retour :

    int, float, string, bool, array, callable, iterable,
    object, mixed, void, never, null, false, true,
    self, parent, static, NomDeClasse, ?Type (nullable)

  6.4 Types d'union et d'intersection (PHP 8.0 / 8.1)
  -----------------------------------------

    function traiter(int|string $valeur): int|false {
        // ...
    }

    function filtrer(Iterator&Countable $collection): void {
        // ...
    }

  6.5 Passage par reference
  -----------------------------------------

    function doubler(&$valeur) {
        $valeur *= 2;
    }

    $x = 5;
    doubler($x);    // $x vaut maintenant 10

  6.6 Nombre variable d'arguments
  -----------------------------------------

    function somme(int ...$nombres): int {
        return array_sum($nombres);
    }

    somme(1, 2, 3);    // 6

  6.7 Arguments nommes (PHP 8.0)
  -----------------------------------------

    function creerUtilisateur(string $nom, int $age, string $ville) {
        // ...
    }

    creerUtilisateur(nom: "Alice", age: 30, ville: "Paris");

  6.8 Fonctions anonymes (closures)
  -----------------------------------------

    $saluer = function($nom) {
        echo "Bonjour $nom";
    };

    $saluer("Alice");

  Capture de variables :

    $message = "Bonjour";
    $fn = function($nom) use ($message) {
        echo "$message $nom";
    };

  6.9 Fonctions flechees (PHP 7.4)
  -----------------------------------------

    $double = fn($x) => $x * 2;

  Les fonctions flechees capturent automatiquement les variables
  du scope parent par valeur.

  6.10 Fonctions de premiere classe (PHP 8.1)
  -----------------------------------------

    $fn = strlen(...);
    echo $fn("test");    // 4

==============================================================================
  7. CHAINES DE CARACTERES
==============================================================================

  7.1 Types de chaines
  -----------------------------------------

  Guillemets simples (pas d'interpolation) :

    $s = 'Texte litteral avec \'apostrophe\'';

  Guillemets doubles (interpolation de variables) :

    $nom = "PHP";
    $s = "Bonjour $nom";
    $s = "Version {$tableau['clef']}";

  Heredoc (interpolation) :

    $texte = <<<EOT
    Texte sur
    plusieurs lignes avec $variable
    EOT;

  Nowdoc (pas d'interpolation) :

    $texte = <<<'EOT'
    Texte litteral sur
    plusieurs lignes
    EOT;

  7.2 Fonctions de chaines principales
  -----------------------------------------

  Fonction             Description
  ---------------      -------------------------------------------
  strlen($s)           Longueur de la chaine
  strtolower($s)       Convertir en minuscules
  strtoupper($s)       Convertir en majuscules
  trim($s)             Supprimer les espaces en debut et fin
  ltrim($s)            Supprimer les espaces en debut
  rtrim($s)            Supprimer les espaces en fin
  substr($s, $d, $l)   Extraire une sous-chaine
  strpos($s, $c)       Position d'une sous-chaine
  str_replace($c,$r,$s) Remplacer dans une chaine
  str_contains($s,$c)  Verifier si contient (PHP 8.0)
  str_starts_with($s,$p) Verifier le debut (PHP 8.0)
  str_ends_with($s,$p)   Verifier la fin (PHP 8.0)
  explode($d, $s)      Decouper en tableau
  implode($d, $a)      Joindre un tableau en chaine
  sprintf($fmt, ...)   Formatage de chaine
  number_format($n)    Formatage de nombre
  str_repeat($s, $n)   Repeter une chaine
  str_pad($s,$l,$c,$t) Completer une chaine
  ucfirst($s)          Premiere lettre en majuscule
  lcfirst($s)          Premiere lettre en minuscule
  ucwords($s)          Premiere lettre de chaque mot en majuscule
  nl2br($s)            Convertir les retours a la ligne en <br>
  htmlspecialchars($s) Echapper les caracteres HTML
  htmlentities($s)     Convertir en entites HTML
  strip_tags($s)       Supprimer les balises HTML
  md5($s)              Hachage MD5
  sha1($s)             Hachage SHA1
  base64_encode($s)    Encodage Base64
  base64_decode($s)    Decodage Base64
  urlencode($s)        Encodage URL
  urldecode($s)        Decodage URL
  json_encode($v)      Encodage JSON
  json_decode($s)      Decodage JSON

  7.3 Expressions regulieres (PCRE)
  -----------------------------------------

    preg_match($motif, $sujet, $resultats)
    preg_match_all($motif, $sujet, $resultats)
    preg_replace($motif, $remplacement, $sujet)
    preg_split($motif, $sujet)

  Exemple :

    if (preg_match('/^[a-z]+$/i', $chaine)) {
        echo "Que des lettres";
    }

==============================================================================
  8. TABLEAUX
==============================================================================

  8.1 Creation de tableaux
  -----------------------------------------

  Tableaux indexes :

    $fruits = ["pomme", "banane", "cerise"];
    $fruits = array("pomme", "banane", "cerise");

  Tableaux associatifs :

    $personne = [
        "nom"  => "Alice",
        "age"  => 30,
        "ville" => "Paris"
    ];

  8.2 Acces aux elements
  -----------------------------------------

    $fruits[0]             Premier element
    $personne["nom"]       Acces par clef
    $fruits[] = "orange";  Ajouter un element

  8.3 Decomposition de tableaux
  -----------------------------------------

    [$a, $b, $c] = [1, 2, 3];
    ["nom" => $nom, "age" => $age] = $personne;
    [, , $troisieme] = $tableau;    // Ignorer des elements

  8.4 Operateur de decomposition (spread)
  -----------------------------------------

    $a = [1, 2, 3];
    $b = [0, ...$a, 4];    // [0, 1, 2, 3, 4]

  8.5 Fonctions de tableaux principales
  -----------------------------------------

  Fonction                  Description
  ----------------------    -------------------------------------------
  count($a)                 Nombre d'elements
  sizeof($a)                Alias de count
  array_push($a, $v)        Ajouter a la fin
  array_pop($a)             Retirer le dernier element
  array_shift($a)           Retirer le premier element
  array_unshift($a, $v)     Ajouter au debut
  array_merge($a, $b)       Fusionner des tableaux
  array_slice($a, $d, $l)   Extraire une portion
  array_splice($a,$d,$l,$r) Remplacer une portion
  array_keys($a)            Obtenir les clefs
  array_values($a)          Obtenir les valeurs
  array_key_exists($k, $a)  Verifier si une clef existe
  in_array($v, $a)          Verifier si une valeur existe
  array_search($v, $a)      Chercher une valeur
  array_unique($a)          Supprimer les doublons
  array_reverse($a)         Inverser l'ordre
  array_flip($a)            Echanger clefs et valeurs
  array_map($fn, $a)        Appliquer une fonction
  array_filter($a, $fn)     Filtrer les elements
  array_reduce($a, $fn)     Reduire a une seule valeur
  array_walk($a, $fn)       Appliquer une fonction (en place)
  array_column($a, $col)    Extraire une colonne
  array_combine($k, $v)     Creer depuis clefs et valeurs
  sort($a)                  Trier par valeur
  rsort($a)                 Trier par valeur (decroissant)
  asort($a)                 Trier par valeur (conserver clefs)
  arsort($a)                Trier par valeur decroissant (clefs)
  ksort($a)                 Trier par clef
  krsort($a)                Trier par clef (decroissant)
  usort($a, $fn)            Trier avec fonction personnalisee
  array_chunk($a, $n)       Decouper en sous-tableaux
  array_fill($d, $n, $v)    Remplir un tableau
  array_diff($a, $b)        Difference entre tableaux
  array_intersect($a, $b)   Intersection de tableaux
  compact('var1', 'var2')   Creer un tableau depuis des variables
  extract($a)               Creer des variables depuis un tableau
  list($a, $b) = $arr       Affecter depuis un tableau

==============================================================================
  9. CLASSES ET OBJETS
==============================================================================

  9.1 Declaration de classe
  -----------------------------------------

    class NomDeClasse {
        // Proprietes
        public string $nom;
        protected int $age;
        private float $solde;

        // Constante de classe
        public const VERSION = "1.0";

        // Constructeur
        public function __construct(string $nom, int $age) {
            $this->nom = $nom;
            $this->age = $age;
        }

        // Methode
        public function saluer(): string {
            return "Bonjour, je suis {$this->nom}";
        }
    }

  9.2 Promotion de proprietes du constructeur (PHP 8.0)
  -----------------------------------------

    class Utilisateur {
        public function __construct(
            public string $nom,
            protected int $age,
            private string $email = ""
        ) {}
    }

  9.3 Instanciation
  -----------------------------------------

    $objet = new NomDeClasse("Alice", 30);
    echo $objet->nom;
    echo $objet->saluer();
    echo NomDeClasse::VERSION;

  9.4 Heritage
  -----------------------------------------

    class Animal {
        public function __construct(
            protected string $nom
        ) {}

        public function parler(): string {
            return "...";
        }
    }

    class Chat extends Animal {
        public function parler(): string {
            return "Miaou";
        }
    }

  PHP ne supporte que l'heritage simple (une seule classe parente).
  Le mot-clef final empeche l'heritage ou la surcharge :

    final class ClasseFinale { }
    class Base {
        final public function methode() { }
    }

  9.5 Classes et methodes abstraites
  -----------------------------------------

    abstract class Forme {
        abstract public function aire(): float;

        public function decrire(): string {
            return "Je suis une forme d'aire " . $this->aire();
        }
    }

    class Cercle extends Forme {
        public function __construct(private float $rayon) {}

        public function aire(): float {
            return M_PI * $this->rayon ** 2;
        }
    }

  9.6 Interfaces
  -----------------------------------------

    interface Affichable {
        public function afficher(): string;
    }

    interface Serialisable {
        public function serialiser(): string;
    }

    class Document implements Affichable, Serialisable {
        public function afficher(): string { /* ... */ }
        public function serialiser(): string { /* ... */ }
    }

  Une classe peut implementer plusieurs interfaces.

  9.7 Traits
  -----------------------------------------

    trait Horodatable {
        public DateTime $creeA;

        public function initialiserDate(): void {
            $this->creeA = new DateTime();
        }
    }

    trait Journalisable {
        public function journal(string $msg): void {
            echo "[LOG] $msg";
        }
    }

    class Commande {
        use Horodatable, Journalisable;
    }

  Resolution de conflit entre traits :

    class MaClasse {
        use TraitA, TraitB {
            TraitA::methode insteadof TraitB;
            TraitB::methode as methodeB;
        }
    }

  9.8 Visibilite
  -----------------------------------------

  Modificateur    Acces
  ------------    -------------------------------------------
  public          Accessible de partout
  protected       Accessible dans la classe et ses enfants
  private         Accessible uniquement dans la classe
  readonly        Propriete en lecture seule (PHP 8.1)

  9.9 Proprietes et methodes statiques
  -----------------------------------------

    class Compteur {
        private static int $nombre = 0;

        public static function incrementer(): void {
            self::$nombre++;
        }

        public static function obtenir(): int {
            return self::$nombre;
        }
    }

    Compteur::incrementer();
    echo Compteur::obtenir();

  9.10 Methodes magiques
  -----------------------------------------

  Methode              Declenchee quand
  ----------------     -------------------------------------------
  __construct()        L'objet est cree
  __destruct()         L'objet est detruit
  __get($nom)          Acces a une propriete inaccessible
  __set($nom, $val)    Ecriture sur une propriete inaccessible
  __isset($nom)        isset() sur une propriete inaccessible
  __unset($nom)        unset() sur une propriete inaccessible
  __call($nom, $args)  Appel d'une methode inaccessible
  __callStatic($n,$a)  Appel statique d'une methode inaccessible
  __toString()         Conversion en chaine
  __invoke(...)        L'objet est utilise comme fonction
  __clone()            L'objet est clone
  __debugInfo()        var_dump() est appele sur l'objet
  __serialize()        Serialisation de l'objet
  __unserialize($d)    Deserialisation de l'objet
  __sleep()            Avant serialisation (obsolete)
  __wakeup()           Apres deserialisation (obsolete)

  9.11 Enumerations (PHP 8.1)
  -----------------------------------------

  Enumeration simple (sans valeur) :

    enum Couleur {
        case Rouge;
        case Vert;
        case Bleu;
    }

    $c = Couleur::Rouge;

  Enumeration avec valeur (backed enum) :

    enum Statut: string {
        case Actif = "actif";
        case Inactif = "inactif";
        case Suspendu = "suspendu";
    }

    $s = Statut::Actif;
    echo $s->value;             // "actif"
    $s = Statut::from("actif"); // Statut::Actif

  Les enumerations peuvent implementer des interfaces et utiliser
  des traits. Elles ne peuvent pas etre instanciees avec new.

  9.12 Classes en lecture seule (PHP 8.2)
  -----------------------------------------

    readonly class Point {
        public function __construct(
            public float $x,
            public float $y
        ) {}
    }

  Toutes les proprietes sont automatiquement readonly.

  10. ESPACES DE NOMS (NAMESPACES)
==============================================================================

  10.1 Declaration
  -----------------------------------------

    namespace App\Models;

    class Utilisateur {
        // ...
    }

  10.2 Utilisation
  -----------------------------------------

    use App\Models\Utilisateur;

  11.1 Exceptions
  -----------------------------------------

    try {
        if ($diviseur == 0) {
            throw new DivisionByZeroError("Division par zero");
        }
        $resultat = $dividende / $diviseur;
    } catch (DivisionByZeroError $e) {
        echo "Erreur : " . $e->getMessage();
    } catch (TypeError | ValueError $e) {
        echo "Erreur de type : " . $e->getMessage();
    } finally {
        echo "Bloc toujours execute";
    }

    +-- Exception
        +-- RuntimeException
        |   +-- OverflowException
        |   +-- UnderflowException
        |   +-- OutOfBoundsException
        +-- LogicException
        |   +-- InvalidArgumentException
        |   +-- LengthException
        |   +-- OutOfRangeException
        |   +-- BadMethodCallException
        |   +-- DomainException
        +-- BadFunctionCallException

  11.3 Exceptions personnalisees
  -----------------------------------------

    class MonException extends RuntimeException {
        public function __construct(
            string $message,
            private array $contexte = [],
            int $code = 0,
            ?\Throwable $precedente = null
        ) {
            parent::__construct($message, $code, $precedente);
        }

        public function getContexte(): array {
            return $this->contexte;
        }
    }
