==============================================================================

  1. Compiler TPCW32 :
     fpc -Mtp TPCW32.PAS -o/tmp/TPCW32

  2. Telecharger corail :
     curl -sL -o /tmp/corail.zip \
       "https://github.com/gladir/corail/archive/refs/heads/master.zip"
     cd /tmp && unzip -q corail.zip

  3. Lancer le test :
     ./TESTS/test_corail.sh /tmp/TPCW32 /tmp/corail-master

  Ou simplement :
     ./TESTS/test_corail.sh
