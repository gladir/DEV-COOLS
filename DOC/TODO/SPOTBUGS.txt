SPOTBUGS.PAS - Clone de SpotBugs - Analyseur statique de bytecode Java (.class)
  Free Pascal (-Mtp)
  Analyse le bytecode des fichiers .class Java pour detecter des bugs potentiels.

  Supporte les fichiers .class individuels, les repertoires et les fichiers .jar.
  Produit des rapports en format texte, XML (compatible Jenkins/SpotBugs) et HTML.
